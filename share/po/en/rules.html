Aim

Simple...

Simple: get diamonds and exit by the door... The number of needed diams is specified in the right panel. The door(s) open(s) when the required number of diams have been taken.

Time is limited, you have to complete the level within the required time.

Sometimes, there are no diams, or not enough...

In this case, you have to make some... Many elements in the game are to be explored, experimented, and maybe will help you to build jewels. Some of these elements may include monsters, special walls... But you will have to discover many others.

I've got the required diams, but... where is the door ?

Some facecious levels do not even have a door... This means that you will probably have to find or build one... Be creative...

Bonus life

A "Bonus life" was originally (in the BoulderDash game) a small level which enabled to gain an extra life if it was completed. If failed, the next level was reached anyway, but with no additional life.

In Rock'n'Roll, there is a "bonus life object", a special diam, which gives you a new life when eat. They are generally found in special levels, small ones like in the original game. You will sometimes have some work to find and take them. If you cannot reach them, taking the regular diam(s) and exiting is still allowed.

Keys and controls

Keyboard and mouse

General controls, when the game is stopped

Action: Control
Scrolling in the level: keyboard arrows, or right mouse + mouse movement
Changing the current level: + and - keys.
Start game: CTRL+J key
Quit: CTRL + Q key.

During game

Action: Control (single player) Multiplayer, player 1 Multiplayer, player 2
Moving: keyboard arrows keyboard arrows Z, W, Q, D keys
Taking a neighboring object without moving: This will also open / close portcullis, or throw a ball, or trigger a lever / button, etc. CTRL + arrow CTRL + arrow ALT + move keys
Abort: will abort the current level, making you lose a life. ESC key F12 key ESC key
Pause: CTRL + P key.
Stop game: CTRL + S key
Quit: CTRL + Q key.
Dropping a bomb: Space key Return key Space key
Selecting another bomb type: TAB key SHIFT key TAB key

Touch devices

General controls

Action: Control
Scrolling in the level: Scroll using the finger.
Zoom: only active when zooming is enabled (see the view menu) pinching with two fingers.

During game

Action: Control
Moving: finger pan action.
Taking a neighboring object without moving: This will also open / close portcullis, or throw a ball, or trigger a lever / button, etc. Touble-tapping, then moving, will trigger this action.
Abort: will abort the current level, making you lose a life. ?? (not possible yet)
Dropping a bomb: Touching / holding the finger, then releasing (without moving) drops a bomb (when the player has bombs).
Selecting another bomb type: ? (not possible yet...)
